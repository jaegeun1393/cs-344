How to run the program?

1. Compile
The way of compile: gcc --std=gnu99 -o movie main.c

How to run the program: ./movie movie_sample_1.csv

2. When you run the program, you have 4 choices.

	1. Show movies released in the specified year
	2. Show highest rated movie for each year
	3. Show the title and year of release of all movies in a specific language
	4. Exit from the program

2-1. once you enter option one, enter the year to see movies that released in the specified year

2-2. Option 2 is showing the highest rated movie for each year

2-3. Option 3 is for searching all movies that has the specific language

2-4. OPtion 4 is for exit the program
